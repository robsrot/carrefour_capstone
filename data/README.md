# Data

Raw and processed data files are **never committed to this repository**.

## Folder structure

```
data/
├── raw/
│   ├── csv/          ← place the original CSV files here
│   └── parquet/      ← auto-generated by the conversion step
└── processed/        ← outputs from pipeline stages
```

## Setup

Place the following files in `data/raw/csv/` after cloning:

| File | Description |
|---|---|
| `ie_maestra_articulos.csv` | Product master — article catalogue |
| `ie_linea_ticket.csv` | Transactional ticket lines (~26 GB) |

Then run the **one-time setup** cell in `notebooks/01_exploration.ipynb` to convert them to Parquet.
